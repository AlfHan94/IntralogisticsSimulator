# Intralogistics Simulator

A mini Warehouse Management System (WMS) built with ASP.NET Core MVC as a learning project and portfolio piece.

## About

This project simulates core warehouse operations including inventory management, order processing, and picking tasks. It demonstrates fundamental concepts used in real-world warehouse management systems.

**Status:** Work in Progress

## Current Features

- Product inventory management with stock levels and locations
- In-memory data storage for rapid prototyping

## Planned Features

- Order creation and management
- Pick task generation and tracking
- Dashboard with warehouse metrics
- Database persistence with Entity Framework Core
- Warehouse process simulation

## Tech Stack

- ASP.NET Core MVC (.NET 10)
- C# 14
- Razor Views
- Bootstrap
- Entity Framework Core (planned)
- SQLite (planned)

## Getting Started

1. Clone this repository
2. Open in Visual Studio 2026
3. Press F5 to run
4. Navigate to `/Products` to view inventory

Developed as an ongoing project to strengthen and apply my skills in modern .NET development, combining my interest in coding and logistics, with the long-term goal of working within intralogistics—especially IT solutions focused on WMS, WCS, and AGV systems, which is where I’d ideally like to end up.
